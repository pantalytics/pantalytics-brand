# Pantalytics Brand Identity

Official brand identity guide for [Pantalytics](https://www.pantalytics.com).

**Live site:** [brand.pantalytics.com](https://brand.pantalytics.com)

## What's inside

- Logo usage guidelines
- Color palette with WCAG contrast ratios
- Typography specimens and scale
- Tone of voice guidelines
- Downloadable brand assets

## Development

```bash
npm install
npm run dev       # http://localhost:4321
npm run build     # Production build
npm run check     # TypeScript validation
```

## Tech stack

- [Astro](https://astro.build/) 5 — static site generator
- Self-hosted fonts (Lexend, Instrument Sans)
- GitHub Pages with custom domain
- Bilingual: Nederlands (default) + English
